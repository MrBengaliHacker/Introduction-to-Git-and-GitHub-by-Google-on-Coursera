# Module 1 Review – Introduction to Version Control

## What I Learned

This module introduced the fundamentals of version control and Git.
It explained why version control is important, how Git enables safe
collaboration, and how changes are tracked using commits.

## Key Skills Practiced

- Understanding version control concepts
- Using Git repositories to track files
- Following the basic Git workflow (init → add → commit)
- Writing meaningful commit messages
- Reviewing file history and changes

## Practice & Assessment

Hands-on practice was completed using Qwiklabs, where Git workflows
were applied in real-world scenarios such as initializing repositories,
adding files, and committing changes.
